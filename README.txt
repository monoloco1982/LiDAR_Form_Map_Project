
PROYECTO: Solicitud de Escaneo LiDAR

REQUISITOS:
- Node.js instalado

INSTRUCCIONES:

1. Instalá dependencias:
   npm install express nodemailer body-parser

2. Pegá tu App Password de Gmail en server.js (línea indicada)

3. Ejecutá el servidor:
   node server.js

4. Abrí tu navegador en:
   http://localhost:3000

¡Listo! Ya podés recibir solicitudes de escaneo vía formulario.
